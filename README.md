# Question

Question & Listing api is the microservice that contains questions. Let's you filter the questions get the content of it. Create new questions.

## Getting Started

Run the application
```
make run
```

Run the application in docker container
```
make up
```

Run the unit tests
```
make unit-test
```

Run all of the tests
```
make test 
```

Run code coverage
```
make code-coverage
```

Run linter
```
make lint
```
